# 长记忆

## 一句话结论
这个文件只放长期稳定、会被反复复用的规则、偏好和背景信息。

## 适用场景
- 当需要快速了解我的长期偏好时
- 当需要组装回答风格或协作方式时
- 当需要给其他记忆内容提供稳定背景时

## 关键信息
- 长期偏好应该放在这里
- 固定工作流程应该放在这里
- 常用术语与约定应该放在这里
- 项目长期背景应该放在这里

## 维护原则
- 内容尽量稳定
- 尽量写成可复用的事实或规则
- 一条信息尽量只表达一个主题
- 频繁变化的内容先放到 `memory/` 里

## 推荐内容类型
- 我的长期偏好
- 固定工作流程
- 常用术语与约定
- 项目长期背景
