# v0.3.1

[English](#english) | [中文](#中文)

## English

### Summary

`v0.3.1` rolls the full current workspace state forward onto the stable release line after `v0.3.0`.

This patch release keeps the `Context Minor GC` closeout line intact while bundling the current Codex-side GC scripts, documentation, reports, and operator-facing repo state into one tagged version.

### Highlights

- includes the full current workspace commit after the `v0.3.0` release line
- keeps `Context Minor GC` as a first-class documented capability and ongoing optimization track
- carries the current Codex-side GC tooling and documentation state into a stable tag
- preserves the `v0.3.0` install and bundle verification path, then advances the stable install target to `v0.3.1`

### Recommended Install

```bash
openclaw plugins install git+https://github.com/redcreen/Unified-Memory-Core.git#v0.3.1
```

## 中文

### 概要

`v0.3.1` 把当前主工作区的完整状态接到 `v0.3.0` 之后的稳定发版线上。

这个 patch 版本保持 `Context Minor GC` 收口成果不变，同时把当前 Codex 侧 GC 脚本、文档、报告和 operator-facing 仓库状态一起收进一个正式 tag。

### 主要内容

- 包含 `v0.3.0` 之后当前主工作区的完整提交状态
- 继续把 `Context Minor GC` 作为一项正式能力和长期优化主线写入公开文档
- 把当前 Codex 侧 GC 工具链与说明文档一起带入稳定 tag
- 保持 `v0.3.0` 已验证过的安装与 bundle verify 路径，同时把稳定安装目标推进到 `v0.3.1`

### 推荐安装方式

```bash
openclaw plugins install git+https://github.com/redcreen/Unified-Memory-Core.git#v0.3.1
```
