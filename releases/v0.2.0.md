# v0.2.0

[English](#english) | [中文](#中文)

## English

### Summary

`v0.2.0` is the first release that closes the Stage 3 and Stage 4 baseline for `unified-memory-core`.

This version moves the project from fact-first governed recall into a releaseable self-learning and policy-adaptation baseline with CLI-first validation.

### Highlights

- Stage 3 self-learning lifecycle baseline closed
- Stage 4 policy adaptation and multi-consumer use closed
- governed policy inputs now project into:
  - OpenClaw
  - Codex
  - generic exports
- new CLI-first repo acceptance flow:
  - `npm run umc:acceptance -- --format markdown`
- new OpenClaw host integration smoke:
  - `npm run umc:openclaw-itest -- --format markdown`
- updated testing docs split:
  - repo acceptance
  - OpenClaw host integration

### Important Note

`v0.2.0` improves release validation and host integration coverage, but it still does not claim to replace or redesign OpenClaw builtin long memory at the host product layer.

The release unit here is still:

- governed plugin-side context assembly
- governed learning lifecycle
- governed policy projection
- CLI-verifiable host integration smoke

### Recommended Install

```bash
openclaw plugins install git+https://github.com/redcreen/Unified-Memory-Core.git#v0.2.0
```

## 中文

### 概要

`v0.2.0` 是 `unified-memory-core` 第一个真正收口 Stage 3 和 Stage 4 基线的版本。

这一版把项目从“事实优先的治理式 recall”推进到“可发布的 self-learning + policy adaptation 基线”，并且把主要验收路径收成了 CLI-first。

### 主要内容

- Stage 3 self-learning lifecycle baseline 已收口
- Stage 4 policy adaptation 与 multi-consumer use 已收口
- governed policy inputs 现在会投影到：
  - OpenClaw
  - Codex
  - generic exports
- 新增仓库内 CLI 验收流：
  - `npm run umc:acceptance -- --format markdown`
- 新增 OpenClaw 宿主集成 smoke：
  - `npm run umc:openclaw-itest -- --format markdown`
- testing 文档已经拆成两层：
  - repo acceptance
  - OpenClaw host integration

### 重要说明

`v0.2.0` 强化了发版验证和宿主集成验证，但它仍然**不表示**要从宿主产品层替代或重写 OpenClaw 内置长期记忆。

这一版真正提供的是：

- 插件侧治理式上下文组装
- 治理式 learning lifecycle
- 治理式 policy projection
- 可通过 CLI 验证的宿主集成 smoke

### 推荐安装方式

```bash
openclaw plugins install git+https://github.com/redcreen/Unified-Memory-Core.git#v0.2.0
```
