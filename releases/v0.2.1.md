# v0.2.1

[English](#english) | [中文](#中文)

## English

### Summary

`v0.2.1` packages the post-`v0.2.0` hardening line into a stable patch release target for `unified-memory-core`.

This release keeps the governed memory model intact while tightening release verification, host deployment confidence, and operator-facing documentation.

### Highlights

- Stage 5 product hardening and independent-operation evidence is now closed
- one-command `release-preflight` now covers repo regression, Stage 5 acceptance, host smoke, real bundle install verification, Markdown link scan, and patch cleanliness
- OpenClaw release bundle build and isolated install verification are both CLI-verifiable
- canonical registry-root policy is explicit and aligned across CLI, docs, and control-surface state
- accepted-action learning now has field-aware Step 47 extraction without reopening the deferred deeper queue

### Recommended Install

```bash
openclaw plugins install git+https://github.com/redcreen/Unified-Memory-Core.git#v0.2.1
```
