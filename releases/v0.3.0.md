# v0.3.0

[English](#english) | [中文](#中文)

## English

### Summary

`v0.3.0` packages the full `Context Minor GC` closeout line into the next stable minor release target for `unified-memory-core`.

This release turns `Context Minor GC` into a clearly documented product capability: in longer conversations near a practical compact danger zone, a topic switch can pull the real prompt back below threshold without requiring an immediate manual `compact`.

### Highlights

- Stage 11 is now formally closed on the OpenClaw host-visible surface
- OpenClaw long-session guarded probes now show real prompt rollback after topic switches instead of only “less growth”
- the near-compaction threshold Docker A/B is now part of the durable evidence surface
- `Context Minor GC` is now documented as one of the long-running optimization tracks, not just a closed milestone
- Codex auto `Context(估算)` header is removed from the default reply path while keeping manual observability commands

### Recommended Install

```bash
openclaw plugins install git+https://github.com/redcreen/Unified-Memory-Core.git#v0.3.0
```

## 中文

### 概要

`v0.3.0` 把 `Context Minor GC` 这一轮完整收口工作打包成 `unified-memory-core` 的下一个稳定 minor 发版目标。

这一版把 `Context Minor GC` 从“内部能力已经可用”进一步推进成“对外可明确说明的产品能力”：更长对话在接近 compact danger zone 时，切题后可以把实际 prompt 拉回阈值下方，而不需要立刻手动 `compact`。

### 主要内容

- Stage 11 已在 OpenClaw 的 host-visible 口径上正式关闭
- OpenClaw 长对话 guarded probe 已证明切题后会有真实 prompt 回落，而不只是“少涨一点”
- near-compaction threshold Docker A/B 已进入 durable 证据面
- `Context Minor GC` 现在被明确写成长期优化主线之一，而不只是一个已关闭的里程碑
- Codex 默认回复路径里的自动 `Context(估算)` header 已移除，但手动观测命令仍保留

### 推荐安装方式

```bash
openclaw plugins install git+https://github.com/redcreen/Unified-Memory-Core.git#v0.3.0
```
