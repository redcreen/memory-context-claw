# Governance System Roadmap

## Goal

Build audit, repair, replay, and regression protection around product artifacts.

## Phases

### Phase 1. Audit surfaces

- audit report schema
- artifact diff views
- baseline comparison rules

### Phase 2. Repair and replay

- repair record model
- replay run model
- non-destructive repair flow

### Phase 3. Regression ownership

- module regression map
- export regression map
- adapter compatibility regression map

## Exit Criteria

- governance actions are attributable and replayable
- repairs can be tracked without hidden mutation
- regression ownership is explicit
