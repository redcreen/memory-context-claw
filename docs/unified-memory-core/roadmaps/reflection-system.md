# Reflection System Roadmap

## Goal

Build a governed candidate-generation layer from normalized artifacts.

## Phases

### Phase 1. Reflection contract

- freeze candidate taxonomy
- freeze reflection question set
- define evidence input contract

### Phase 2. Candidate MVP

- event labeling
- pattern extraction
- fact / rule / habit candidate builders

### Phase 3. Evidence and reporting

- confidence scoring
- reflection run summaries
- replay-friendly outputs

## Exit Criteria

- normalized artifacts can produce reviewable candidates
- evidence scoring is deterministic enough for tests
- reflection outputs are registry-ready
