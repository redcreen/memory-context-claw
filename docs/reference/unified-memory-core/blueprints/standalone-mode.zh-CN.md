# Standalone Mode Blueprint

[English](standalone-mode.md) | [中文](standalone-mode.zh-CN.md)

## MVP 形态

- standalone command router
- source / reflect / export / govern command families
- dry-run 和 inspect outputs
- 面向 scheduled jobs 的调用方式

## 第一批产出

- command reference draft
- CLI artifact path conventions
- standalone run report shape

## 关键风险

- command surface 过度碎片化
- 隐式 adapter coupling
- standalone outputs 偏离 embedded contracts
