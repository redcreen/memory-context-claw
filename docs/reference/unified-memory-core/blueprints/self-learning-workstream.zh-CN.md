# Self-Learning Workstream Blueprint

[English](self-learning-workstream.md) | [中文](self-learning-workstream.zh-CN.md)

## MVP 形态

- daily reflection runner
- candidate artifact pipeline
- promotion / decay evaluation hooks
- policy adaptation inputs

## 第一批产出

- reflection report
- candidate artifact set
- promotion review report
- policy-input artifact draft

## 关键风险

- candidate 噪音爆炸
- evidence attribution 不足
- reflection 逻辑滑向自由总结
